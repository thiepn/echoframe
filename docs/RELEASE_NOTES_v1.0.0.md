# ECHOFRAME: LAST SIGNAL — Version 1.0

ECHOFRAME: LAST SIGNAL is a deterministic desktop keyboard-and-mouse Phaser 3 action game built around recording and replaying the Warden's recent actions as friendly Echoes.

## Version 1.0 includes

- Complete six-combat/elite run structure, recovery chamber, seven upgrade choices, and the three-phase Null Architect boss.
- First Signal production tutorial with persistent completion and Archive replay.
- Twenty-four canonical upgrades, deterministic arenas and encounters, immutable score ledger, combo system, progression, Archive, Statistics, Settings, Credits, and Results.
- Keyboard and pointer rebinding with primary/secondary bindings, conflict rejection, persistence, and immediate runtime refresh.
- Accessibility presets covering motion, flashes, particles, contrast, outlines, locator, HUD opacity, damage numbers, and aim line.
- Local-only schema-2 save data with Phase 9 migration, validated import/export, no analytics, and no telemetry.

## Supported release target

- Desktop Chromium and Firefox
- Keyboard and mouse
- Minimum CSS viewport 1024 × 576
- Static hosting at a root path or repository subpath

Mobile, touch, gamepad, localization, online services, cloud saves, leaderboards, and endless mode are outside Version 1.0.
